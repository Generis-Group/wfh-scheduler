# Generis Reporting App Visual Reference Prompt Pack

These prompts generated the three redesign boards used as visual references for the UI refresh. The images are reference-only and should not be embedded as product UI.

## Employee Board

Design a minimal professional dark-mode operational web app for Generis daily reporting, focused on the employee view. Fix these issues: too many equal-weight borders, cramped row action buttons against table edges, flat dark surfaces, inconsistent spacing, raw textarea-heavy cards, spreadsheet-like tables, weak typography hierarchy, awkward logo/nav balance, and disconnected primary actions. Use the official Generis teal wordmark feel, restrained blue accents, neutral dark surfaces, no decorative gradients, no marketing hero, no stock imagery, no bokeh/orbs. Show Daily Entry, Reports workspace, Settings, and Account. Daily Entry has a focused report editor with date/status/action header, activity table with padded fixed actions column and subtle ghost more button, sync controls as quiet secondary actions, summary and blockers in a balanced right panel, save draft and submit grouped with report state. Reports is the workspace for drafts/submitted history, filters, open-date control, report list, selected report detail panel, submit draft/edit actions, comments, and revisions. Settings and Account are full-width operational pages with profile, security, integrations, notifications, and appearance cards. Use neutral placeholder labels only.

## Admin / Reviewer Board

Design a minimal professional dark-mode operational web app for Generis daily reporting, focused on the reviewer/admin view. Fix these issues: too many equal-weight borders, cramped row action buttons, flat dark surfaces, inconsistent spacing, raw form-heavy cards, spreadsheet-like tables, weak typography hierarchy, awkward logo/nav balance, and disconnected primary actions. Use official Generis teal wordmark feel, restrained blue accents, neutral dark surfaces, soft surface layering, fewer outlines, no decorative gradients, no marketing styling. Show Review Dashboard, Employees, Admin Settings, and Account. Review Dashboard has date/filter toolbar, compact KPI row, coverage table, selected report detail panel, comments, source-mix and blocker trend charts, bulk actions, and manual email digest action. Employees has user table, invite/create form, role/status controls, reset temporary password, and company settings panel. Settings and Account have admin email digest status, integrations, profile, security, appearance, and company configuration. Use neutral placeholders only.

## Login / Preview Board

Design a minimal professional authentication and preview entry experience for the Generis Daily Reporting App. Match the redesigned operational SaaS style used by employee and admin pages. Use official Generis teal logo feel, restrained blue accent, neutral light and dark modes, soft surface layering, minimal borders, no decorative gradients, no marketing hero, no large illustration, no stock imagery. Show Login light mode, Login dark mode, Preview chooser light mode, and Preview chooser dark mode. Login includes company-only @generisgp.com email/password credentials, Google and Atlassian connect buttons, clear small auth error area, and temporary preview bypass buttons on the login page. Preview chooser includes View as Employee and View as Admin/Reviewer and notes that preview is development-only.

# Work From Home Scheduler Documentation

This directory contains documentation for the Generis work from home scheduler and daily reporting app.

- [User Guide](./user-guide.md): how employees, reviewers, and admins use the app.
- [Developer Guide](./developer-guide.md): how the app is built, configured, tested, and extended.

The app combines daily work reporting, work-location planning, team review, source imports, AI-assisted summaries, and admin tools.
